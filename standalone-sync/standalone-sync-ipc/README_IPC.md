# CODESYS Persistent UI / File IPC Sync Fork

Этот каталог содержит автономный форк утилиты синхронизации, оптимизированный для работы в режиме запущенной среды **Abak.IDE (CODESYS V3.5)** с открытым GUI-проектом.

## Структура файлов форка

1. **`ipc_listener.py`** — Серверный скрипт, запускаемый непосредственно **внутри среды Abak.IDE** (*Tools -> Scripting -> Run Script*). 
   * Он использует асинхронный .NET-таймер (`System.Windows.Forms.Timer`) для регулярного сканирования запросов в фоновом режиме на UI-потоке, при этом интерфейс среды остается полностью отзывчивым.
   * Принимает команды и выполняет их в оперативной памяти над `projects.primary`.
2. **`gui_manager_ipc.py`** — Графический клиент (PyQt6/PyQt5) для разработчиков. Позволяет отправлять команды в среду, просматривать логи выполнения в реальном времени и управлять версионированием через встроенную вкладку Git.
3. **`sync_ipc_cli.py`** — Интерфейс командной строки (CLI) клиента. **Главный инструмент автоматизации для ИИ-агента (Antigravity)**. Позволяет агенту взаимодействовать с CODESYS без участия человека.
4. **`manage_ipc.bat`** — Скрипт для быстрого запуска графического клиента.

## Руководство для ИИ-агента (Antigravity) по работе с открытой средой

Если пользователь просит вас синхронизировать, импортировать, экспортировать или скомпилировать проект в запущенной Abak.IDE, **вы можете делать это автономно** через инструмент `run_command` с помощью `sync_ipc_cli.py`.

### 1. Экспорт исходников (Export)
Чтобы выгрузить актуальный Structured Text (.st) код из открытого проекта CODESYS на диск:
```bash
python standalone-sync-ipc/sync_ipc_cli.py --action export
```
* **Результат**: Лог выполнения покажет все экспортированные файлы в формате `file:///c:/...`. 
* **Поведение агента**: Прочитайте лог, найдите интересующий вас файл по ссылке `file:///` и прочитайте его содержимое с помощью инструмента `view_file`.

### 2. Импорт изменений (Import)
После того как вы изменили один или несколько `.st` файлов на диске, отправьте их обратно в открытый проект CODESYS:
```bash
python standalone-sync-ipc/sync_ipc_cli.py --action import
```
* **Результат**: Изменения мгновенно применятся к открытым POU прямо на экране разработчика в Abak.IDE и проект сохранится.

### 3. Компиляция проекта (Compile)
Чтобы проверить проект на ошибки компиляции внутри среды:
```bash
python standalone-sync-ipc/sync_ipc_cli.py --action compile
```
* **Результат**: Скрипт вернет лог компиляции со всеми предупреждениями и ошибками (Warnings/Errors). Вы сможете проанализировать ошибки и исправить их.

### 4. Остановка сервера (Stop)
Для деактивации таймера в среде:
```bash
python standalone-sync-ipc/sync_ipc_cli.py --action stop
```

## Эргономика для разработчика

* **Быстрый доступ к исходникам**: В GUI-клиенте добавлена кнопка **📂 Открыть папку ST в Проводнике**, которая мгновенно открывает папку с исходными кодами в проводнике Windows.
* **Интерактивные логи**: Все пути к файлам `file:///` в логах клиента автоматически преобразуются в кликабельные бирюзовые ссылки. Нажатие на ссылку открывает файл в вашем установленном редакторе по умолчанию (например, VS Code).
